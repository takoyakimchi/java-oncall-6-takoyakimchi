## 입출력
### 월, 시작 요일 입력
* 예시 `5,월`
* 이 경우 5월 1일은 월요일이다.
### 평일 비상근무 사원 이름 입력
* `,`로 구분
* 순번대로 입력
### 휴일 비상근무 사원 이름 입력
* `,`로 구분
* 순번대로 입력
### 근무 배치 결과 출력
* 예시
```
5월 1일 월 준팍
5월 2일 화 도밥
5월 3일 수 고니
5월 4일 목 수아
5월 5일 금(휴일) 루루
...
```

## 비상 근무일 배정 로직
* 평일 1회, 휴일 1회 배정 필수
  * 평일과 휴일 근무자명 일치하는지 체크
  * 평일이나 휴일에 2회 배정된 경우(중복)가 있을 시 오류
* 연속 2일 근무할 수 없다
  * 연속 2일 근무하는 경우 다음 근무자와 순서 바꾼다
* 닉네임은 최대 5자
* 근무자 인원은 5명~35명
* 휴일
  * 토요일, 일요일
  * 1월 1일
  * 3월 1일
  * 5월 5일
  * 6월 6일
  * 8월 15일
  * 10월 3일
  * 10월 9일
  * 12월 25일
